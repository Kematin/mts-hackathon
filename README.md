# МТС True Tech Hack 2026

## Запуск
1. Склонируйте репозиторий и перейдите в него
   ```bash
   git clone https://git.truetecharena.ru/tta true-tech-hack2026-localscript/asdasdasd/task-repo.git 
   cd true-tech-hack2026-localscript/asdasdasd
   ```
2. Создайте .env и вставте в него переменные из env.example
3. Запустите приложение через docker compose \
   (должен быть установлен [движок](https://docs.docker.com/engine/install/))
   ```bash
   docker compose up -d --build
   ```
